# ✍️Commands & Permissions🦾

## Commands ✍️

`/trademanager` - in-game GUI editor

## Permissions 🦾

* `trademanager.*` - All permissions node
* `trademanager.admin` - Access to all TM commands
* `trademanager.alerts` - Update notifcations (OP got these by default)
